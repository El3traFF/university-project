# University Management System (Консольна система управління університетом)

Консольний застосунок на Java для управління студентами, викладачами, курсами та
зарахуваннями на курси. Проєкт реалізує CRUD-операції, ООП (успадкування, інтерфейси,
enum), обробку винятків та звіти/пошук.

## Структура проєкту

```
src/university/
 ├── Main.java                 // головний клас з консольним меню
 ├── entities/
 │    ├── Person.java          // батьківський абстрактний клас
 │    ├── Student.java         // extends Person
 │    ├── Teacher.java         // extends Person
 │    ├── Course.java
 │    └── Enrollment.java      // implements Payable
 ├── enums/
 │    ├── Grade.java           // A, B, C, D, F, NA
 │    ├── StudentStatus.java   // ACTIVE, ON_LEAVE, EXPELLED, GRADUATED
 │    └── TeacherPosition.java // ASSISTANT, LECTURER, PROFESSOR
 ├── interfaces/
 │    └── Payable.java
 ├── services/
 │    ├── StudentService.java
 │    ├── TeacherService.java
 │    ├── CourseService.java
 │    └── EnrollmentService.java
 └── util/
      └── GPAUtils.java        // обчислення GPA, сортування (bubble/selection)
```

## Як зібрати та запустити
```bash
# Компіляція
javac -d out $(find src -name "*.java")

# Запуск (важливо явно вказати UTF-8, щоб коректно відображалась кирилиця)
java -Dfile.encoding=UTF-8 -Dstdout.encoding=UTF-8 -cp out university.Main
```

Якщо у консолі кирилиця відображається некоректно (питання замість літер), переконайтесь,
що термінал/IDE налаштовані на кодування UTF-8, або запустіть з прапорцями вище.

## Функціонал

- **Студенти**: додавання, перегляд, оновлення, видалення, зміна статусу,
  фільтр за статусом/роком, сортування за ПІБ (bubble sort), пошук.
- **Викладачі**: CRUD, фільтр за посадою.
- **Курси**: CRUD, фільтр за викладачем/кредитами.
- **Зарахування**: створення (studentId + courseId + semester, grade=NA, paid=false),
  виставлення оцінки, позначення оплати, перегляд зарахувань студента з GPA, транскрипт.
- **Звіти/Пошук**: пошук студента за ПІБ/email, список неоплачених курсів,
  середній GPA по курсу/семестру, топ-N студентів за GPA (selection sort).

## Дані

Усі дані зберігаються в пам'яті програми в масивах, що динамічно розширюються
(`Arrays.copyOf`), без використання `ArrayList` — відповідно до вимог завдання.
При старті програма заповнюється кількома тестовими записами (студенти, викладачі,
курси, зарахування), щоб одразу можна було перевірити всі сценарії.

## Обробка помилок

Некоректний ввід (текст замість числа, невірний email, рік < 1, невідомий enum-статус
тощо) призводить до `IllegalArgumentException` з зрозумілим повідомленням, яке
перехоплюється в `Main` і виводиться користувачу без аварійного завершення програми.
